Step 5 — React Components (client/src/components/fms/)
Create the directory client/src/components/fms/ and add the following three component files. These use the existing shadcn/ui components already installed in the project.

🎨  Design principles for all FMS components:
— Use existing shadcn/ui components: Button, Dialog, Input, Label, Select, Badge, Card
— Use TanStack Query (useQuery / useMutation) for all data fetching — match the pattern in existing hooks
— Use the existing useToast hook for success/error feedback
— Tailwind only — no inline styles
— Mobile-first: all layouts must work at 375px width
